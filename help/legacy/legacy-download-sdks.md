## Mídia de transmissão e a extensão do Adobe Medium Analytics

A extensão Adobe Medium Analytics adiciona o principal SDK de mídia JavaScript (SDK do Media 2.x) para áudio e vídeo. Esta extensão fornece a funcionalidade de adicionar a instância do rastreador `MediaHeartbeat` a um site ou projeto de Coleção de dados.

A coleta de dados do Adobe com a extensão do Media Analytics tem os seguintes requisitos:
* Você deve ser um cliente da Adobe Experience Cloud.
* Você deve implantar o código de incorporação de Coleção de dados ou DTM nas páginas da Web.
* Você deve configurar o [Extensão do Analytics](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/analytics/overview.html?lang=pt-BR).
* Você deve configurar o [Extensão de inicialização do Adobe Medium Analytics for Audio and Video](https://aep-sdks.gitbook.io/docs/using-mobile-extensions/adobe-media-analytics).
* Você deve configurar o [Extensão Experience Cloud ID](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/id-service/overview.html?lang=pt-BR).
